Você está subindo uma escada. São necessários n degraus para chegar ao topo.

Cada vez você pode subir 1 ou 2 degraus. De quantas maneiras distintas você pode subir até o topo?



* Exemplo 1:

Entrada: n = 2

Saída: 2

Explicação: Há duas maneiras de subir até o topo.
1. 1 passo + 1 passo
2. 2 passos

* Exemplo 2:

Entrada: n = 3

Saída: 3

Explicação: Há três maneiras de subir até o topo.
1. 1 passo + 1 passo + 1 passo
2. 1 passo + 2 passos
3. 2 passos + 1 passo
